# M62 WEB TV - Minimum Graphic Checklist

Goal: launch fast without waiting for full branding pack.

## Priority High (Before Go-Live)

1. Primary Logo
- Usage: header logo, browser identity.
- File: assets/images/logo-main.png
- Size: 512x512 (master), export 200x200 and 120x120
- Format: PNG (transparent background)

2. Favicon
- Usage: browser tab and bookmarks.
- File: favicon.ico (root), assets/icons/favicon-32.png, assets/icons/favicon-16.png
- Size: 32x32 and 16x16
- Format: ICO + PNG

3. Hero Banner
- Usage: homepage hero section.
- File: assets/images/hero-banner.jpg
- Size: 1920x1080
- Format: JPG (quality 75-85)

4. Default News Cover Placeholder
- Usage: when a news item has no cover image.
- File: assets/images/news-default.jpg
- Size: 1200x675
- Format: JPG

5. Default Video Thumbnail Placeholder
- Usage: when video has no thumbnail.
- File: assets/images/video-default.jpg
- Size: 1280x720
- Format: JPG

## Priority Medium (Week 2)

1. Category Covers (News)
- Categories: politics, economy, sports, religion, education, entertainment.
- Path: assets/images/categories/
- Size: 1200x675 each
- Format: JPG

2. Program Cards Background Set
- For: business, religion, entertainment, education, talk show, history, kids, Q&A, law.
- Path: assets/images/programs/
- Size: 800x450 each
- Format: JPG/WEBP

3. Social Share Preview Image (Open Graph)
- Usage: Facebook/X/WhatsApp link preview.
- File: assets/images/social-preview.jpg
- Size: 1200x630
- Format: JPG

4. Team / About Photos
- Path: assets/images/team/
- Size: 800x800
- Format: JPG/WEBP

## Priority Low (After Launch)

1. Custom Icons Set
- For categories and admin shortcuts.
- Path: assets/icons/custom/
- Format: SVG preferred

2. Motion Graphics / Intro Video
- For opening or social campaigns.
- Path: assets/videos/branding/
- Format: MP4 (H.264)

3. Background Patterns / Decorative Shapes
- For premium visual polish.
- Path: assets/images/patterns/
- Format: SVG/PNG

## Technical Rules

1. Naming convention
- Use lowercase and hyphen names.
- Example: hero-banner.jpg, news-default.jpg

2. Compression
- JPG target: less than 300 KB per image where possible.
- PNG target: less than 200 KB for icons/logos.

3. Safe text area
- Keep important text inside center 80% of image.

4. Copyright safety
- Use licensed or original assets only.

## Fast Launch Decision

If time is short, launch with only Priority High assets.
Priority Medium and Low can be rolled out post-launch without breaking existing pages.
